# 番茄钟 (Pomodoro Timer) — Web 版

一个精美的网页番茄钟，纯 HTML/CSS/JS 实现，无需任何依赖。

## 功能

- 🍅 SVG 圆形进度环，平滑动画
- ☕ 短休 / 🌴 长休自动切换
- 🔔 Web Audio 风铃音效 + 毛玻璃弹窗提醒
- 🎨 三套配色自动切换（赤陶/鼠尾草/金色）
- ⚙ 自定义工作时长、休息时长、长休间隔
- 💾 localStorage 本地持久化配置
- ⌨ 键盘快捷键（空格/P/R/1/2/3/Esc）
- 📌 桌面双击 `番茄钟.bat` 即可在浏览器打开

## 运行

直接在浏览器打开 `pomodoro.html`，或双击 `番茄钟.bat`。

## 技术栈

- HTML/CSS/JS（单文件，零依赖）
- SVG stroke-dasharray 圆环动画
- Web Audio API 音效
- localStorage 配置持久化

## 分支

- `main` — Python tkinter 原版
- `webV` — 网页版（当前分支）
